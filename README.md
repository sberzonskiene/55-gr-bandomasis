# 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo   

    git clone https://github.com/sberzonskiene/55-gr-bandomasis 

2. Install main NPM packages

    npm i

3. Install sub-directories NPM packages

    npm run install-all

4. Run the servers

    npm run dev
